## ⚠️ Последний шаг: Настройка DNS на Reg.ru

### Зачем это нужно?

CNAME файл говорит GitHub: "этот сайт для домена yaroslavtarasov.com"

Но нужно настроить сам домен, чтобы он **указывал на GitHub Pages**.

---

## 📋 Пошаговая инструкция (5 минут)

### Шаг 1: Зайти в панель Reg.ru

1. Откройте: **https://www.reg.ru/user/account/**
2. Войдите в аккаунт
3. Найдите домен **yaroslavtarasov.com**

### Шаг 2: Открыть настройки DNS

Найдите один из вариантов:
- "Управление зоной DNS"
- "DNS-серверы и зона"
- "Редактировать зону"

### Шаг 3: Добавить A-записи для GitHub Pages

Добавьте **4 A-записи**:

```
┌─────────────────────────────────────────────┐

│ Тип: A                                      │
│ Хост: @ (или пусто, или yaroslavtarasov.com)│
│ IP-адрес: 185.199.108.153                   │
│ TTL: 3600                                   │

┌─────────────────────────────────────────────┐
│ Запись 2:                                   │
│ Тип: A                                      │

│ IP-адрес: 185.199.109.153                   │

┌─────────────────────────────────────────────┐
│ Запись 3:                                   │
│ Тип: A                                      │
│ Хост: @                                     │

│ TTL: 3600                                   │
└─────────────────────────────────────────────┘
```

### Шаг 4: (Опционально) Добавить CNAME для www

Если хотите, чтобы www.yaroslavtarasov.com тоже работал:

```
Тип: CNAME
Хост: www
Значение: yatarasov724.github.io.
TTL: 3600
```

⚠️ **ВАЖНО:** В конце должна быть точка: `yatarasov724.github.io.`

### Шаг 5: Сохранить изменения

## 🔍 Как проверить, что DNS настроен

### Способ 1: Online инструмент (самый простой)

Откройте: **https://www.whatsmydns.net/**

1. Введите: `yaroslavtarasov.com`
2. Выберите тип: **A**
3. Нажмите **Search**

**Должно показать:** 4 IP-адреса GitHub (185.199.*)

Если в России показывает эти IP - DNS работает! ✅

### Способ 2: Через терминал (для продвинутых)

```bash
# Проверить A-записи
dig yaroslavtarasov.com +short

# Должно вывести:
# 185.199.108.153
# 185.199.109.153
# 185.199.110.153
# 185.199.111.153
```

## 🐛 Возможные проблемы

### Проблема: "DNS_PROBE_FINISHED_NXDOMAIN"

**Причина:** DNS записи еще не применились или неправильные

**Решение:**
1. Подождите 1-2 часа
2. Проверьте DNS через whatsmydns.net
3. Убедитесь, что добавлены все 4 A-записи

### Проблема: 404 - File not found

**Причина:** DNS работает, но GitHub Pages не видит домен

**Решение:**
1. Откройте: https://github.com/yatarasov724/portfolio/settings/pages
2. Убедитесь, что в "Custom domain" указан yaroslavtarasov.com
3. Если нет - добавьте вручную и сохраните

### Проблема: ERR_SSL_PROTOCOL_ERROR

**Причина:** SSL сертификат еще не выдан

**Решение:**
1. Используйте `http://` вместо `https://` временно
2. Подождите 24 часа
3. После выдачи сертификата включите "Enforce HTTPS"
