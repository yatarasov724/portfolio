## Что нужно сделать перед первым деплоем

1. **Закоммить все изменения:**
   ```bash
   git add .
   git commit -m "Готово к деплою: оптимизация и улучшения"
   ```

2. **Создать репозиторий на GitHub** (см. DEPLOY.md)

3. **Отправить код на GitHub:**
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
   git push -u origin main
   ```

4. **Зайти на Vercel и задеплоить** (см. QUICK_START.md)
